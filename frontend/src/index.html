Network Validation Settings

Configure how IP addresses are validated for private networks. These settings affect both frontend validation and user experience.

When enabled, allows any valid IPv4 address in private network configurations. When disabled (recommended), only private IP ranges (192.168.x.x, 10.x.x.x, 172.16-31.x.x) are allowed.

Security Warning

Allowing public IP addresses in private networks may create connectivity issues and security concerns. This option is intended for advanced users with specific network requirements.
